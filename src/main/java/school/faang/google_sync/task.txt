Описание

Вы разработчик в Google, и ваша задача — создать приложение для автоматической загрузки фотографий в Google Photos.
Приложение должно загружать фотографии из папки на компьютере пользователя на сервер Google Photos.
Если в папке нет новых фотографий, приложение должно приостановить работу (перейти в режим ожидания) и возобновить загрузку,
когда новые фотографии будут добавлены в папку.

Требования:
1. Создайте класс GooglePhotosAutoUploader, который будет использовать объект lock для синхронизации потоков.

2. Создайте список photosToUpload, в котором будут храниться пути к фотографиям (строки), которые нужно загрузить.

3. Реализуйте метод startAutoUpload(), который будет:

Проверять, есть ли фотографии для загрузки.

Если список пуст, программа должна ждать, пока не появятся новые фотографии.

Если фотографии добавлены, метод должен их загрузить с помощью вызова uploadPhotos().

4. Реализуйте метод onNewPhotoAdded(String photoPath), который:

Добавляет новый путь к фотографии в список photosToUpload.

Уведомляет поток, который находится в режиме ожидания, о том, что появились новые фотографии

5. Создайте метод uploadPhotos(), который:

Загружает фотографии на сервер (можно просто имитировать этот процесс, выводя сообщения на экран).

После загрузки удаляет фотографии из списка photosToUpload.

6. Создайте класс с методом main():

Создайте объект класса GooglePhotosAutoUploader.

Создайте два потока: один поток для автоматической загрузки фотографий, другой для добавления новых фотографий.

Реализуйте логику в методах run() для каждого потока и запустите их.
