В семье Уизли всегда полно домашних хлопот, и чтобы справиться с этим волшебным хаосом, Перси решил автоматизировать выполнение задач. Его идея — использовать пул потоков с помощью метода newCachedThreadPool(), чтобы поручить выполнение всех дел сразу нескольким потокам.

Требования:

1. Создайте класс Chore, который будет реализовывать интерфейс Runnable.

Этот класс будет описывать каждую домашнюю задачу, которую нужно выполнить.

2. Добавьте поле chore типа String в класс Chore, чтобы хранить название задачи (например, "помыть посуду", "подмести пол"). Создайте конструктор, который принимает название задачи как параметр.

3. В методе run() класса Chore опишите логику выполнения задачи.

Используйте Thread.currentThread().getName() для вывода имени потока, который выполняет задачу, и Thread.sleep() для имитации выполнения задачи.

4. В классе WeasleyFamily создайте массив chores, содержащий список задач, которые нужно выполнить.

Например, ["помыть посуду", "подмести пол", "приготовить ужин"].

5. Создайте пул потоков с помощью Executors.newCachedThreadPool().

Этот пул потоков автоматически создаст новые потоки по мере необходимости и переиспользует старые для экономии ресурсов.

6. Добавьте задачи в пул потоков.

Создайте объект класса Chore для каждой задачи из массива chores и добавьте его в пул потоков

7. Завершите работу пула потоков после выполнения всех задач.
