У Walmart есть несколько касс, и каждая из них обслуживает покупателей. Каждый покупатель представляет собой отдельный поток. Напишите программу, которая использует наследование от класса Thread для моделирования работы касс в Walmart.

Когда покупатель приходит в Walmart, он случайным образом выбирает одну из касс и становится в очередь. Покупатель ждёт, пока кассир закончит обслуживание текущего покупателя, а затем начинается его обслуживание.

Требования задачи:

1. Создайте класс CashierThread, который наследуется от класса Thread. Объявите поля:

cashierId — идентификатор кассира.

customerItems — массив товаров покупателя (количество товаров).

2. Создайте конструктор для класса CashierThread, который будет принимать идентификатор кассира и массив товаров покупателя.

3. Реализуйте метод run():

Симулируйте обработку товаров кассиром: перебирайте массив товаров покупателя, выводя информацию о каждом товаре.

После обработки всех товаров покупателя выведите сообщение о количестве товаров и общей стоимости покупок.

4. В методе main():

Создайте двумерный массив customers, где каждый элемент представляет массив товаров для одного покупателя (например, корзина покупок).

Создайте массив cashiers для хранения объектов класса CashierThread.

5. Распределите покупателей между кассирами:

В цикле создайте объекты класса CashierThread для каждого покупателя и добавьте их в массив cashiers.

Каждого покупателя случайным образом распределяйте между кассирами.

6. Запустите все потоки кассиров: В цикле вызовите метод start() для каждого объекта CashierThread из массива cashiers.

7. Дождитесь завершения всех потоков и выведите на экран информацию о количестве обработанных товаров и общей стоимости покупок для каждого покупателя.
