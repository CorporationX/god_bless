На обеденном перерыве женщина из отдела кадров попросила вас написать программу, которая из огромного списка людей выведет имена, фамилии и места их работы. Но поскольку список очень большой (например, 10 000 человек), чтобы ускорить процесс, нужно обработать его в несколько потоков, распределяя работу между ними.

Требования задачи:

1. Создайте класс Person с полями:

name — имя человека.

surname — фамилия.

age — возраст.

workplace — место работы.

2. Создайте класс PersonInfoPrinter, который реализует интерфейс Runnable. Этот класс будет выводить информацию о человеке (имя, фамилия, место работы) для каждого человека из переданного списка.

3. В методе main класса Main создайте список объектов класса Person, содержащий данные о 10 000 людях. Поля могут быть заполнены простыми значениями, например, с использованием индексов для генерации данных.

4. Разбейте работу на несколько потоков:

В методе main создайте объект ExecutorService с помощью метода newFixedThreadPool класса Executors.

Определите, сколько потоков будет использоваться для обработки списка людей.

5. Разделите список на несколько частей:

Разбейте список на части, количество которых должно соответствовать количеству потоков.

Для каждой части списка создайте объект PersonInfoPrinter и передайте ему подсписок людей для обработки.

6. Запустите задачи в пуле потоков.

7. Завершите работу пула потоков. Вызовите метод shutdown() у объекта ExecutorService, чтобы прекратить приём новых задач после того, как все задания будут переданы.

8. Дождитесь завершения всех задач. Используйте метод awaitTermination(), чтобы дождаться завершения всех потоков.

9. Выведите сообщение о завершении работы программы в консоль после того, как все потоки завершат свою работу.
