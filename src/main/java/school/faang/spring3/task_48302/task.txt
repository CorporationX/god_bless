Шелдон, Леонард, Говард и Раджеш работают над научным проектом и решают разделить задания между собой для ускорения работы. Ваша задача — написать программу, которая создаёт пул из четырёх потоков с помощью класса ExecutorService и распределяет выполнение задач между этими персонажами:

Шелдон — подготовка теории.

Леонард — моделирование эксперимента.

Говард — разработка инструментов.

Раджеш — анализ данных.

Требования:

1. Создайте класс Task, который будет реализовывать интерфейс Runnable. В этом классе будут два поля:

name (имя исполнителя)

task (задача, которую нужно выполнить).

Не забудьте создать конструктор для инициализации этих полей.

2. Реализуйте метод run в классе Task. Этот метод должен:

Выводить сообщение о начале выполнения задачи.

Имитировать выполнение задачи с помощью Thread.sleep()

В случае, если задача будет прервана, вывести сообщение об этом и завершить выполнение метода.

3. Создайте класс BigBangTheory с методом main(). В нём создайте объект ExecutorService на 4 потока.

4. Создайте четыре объекта класса Task, передавая в каждый объект имя и соответствующую задачу.

5. Добавьте каждый объект Task в ExecutorService для выполнения задачи

6. Выведите сообщение по завершению выполнения каждой задачи

7. После завершения всех задач остановите пул потоков
