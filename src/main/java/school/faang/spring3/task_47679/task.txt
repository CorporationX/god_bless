Вы являетесь программистом в компании, которая разрабатывает игру про трёх поросят. В игре нужно иметь возможность управлять каждым поросенком отдельно. Для этого требуется создать три потока, каждый из которых будет управлять одним из поросят. Необходимо наследоваться от класса Thread и переопределить метод run() для каждого потока так, чтобы каждый из них мог управлять своим поросенком.

Требования:

1. Создайте класс PigThread, который наследуется от класса Thread. Поля:

pigName — имя поросенка (например, Ниф-Ниф, Нуф-Нуф, Наф-Наф).

material — материал, из которого поросенок строит дом (солома, палки, кирпичи).

Этот класс будет базовым для потоков поросят.

2. Создайте конструктор в классе PigThread, который принимает имя поросенка и материал строительства. Добавьте геттеры для этих полей.

3. Создайте классы Pig1Thread, Pig2Thread, Pig3Thread, которые наследуются от PigThread. В каждом классе реализуйте логику строительства дома из нужного материала.

4. Реализуйте метод run() в каждом классе Pig1Thread, Pig2Thread и Pig3Thread:

Поросята должны строить дом, выводя на экран информацию о том, что они используют соответствующий материал.

Добавьте задержку для симуляции времени строительства.

5. В методе main() класса ThreeLittlePigs:

Создайте три объекта потоков для каждого поросенка (Ниф-Ниф, Нуф-Нуф, Наф-Наф), каждый из которых использует свой материал.

Запустите все потоки.

После завершения всех потоков выведите на экран сообщение «Игра завершена!».
