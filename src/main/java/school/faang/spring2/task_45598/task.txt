Условие задачи
Описание:
Вам нужно написать программу для преобразования квадратной матрицы (двумерного массива) с помощью функционального интерфейса. Программа должна уметь переворачивать матрицу по горизонтали и по вертикали, а также использовать интерфейс для различных трансформаций.

Требования:
1️⃣ Создайте функциональный интерфейс MatrixTransformer, который:

принимает два индекса ячейки матрицы (индексы строки и столбца),

возвращает объект Coordinates, содержащий новые индексы для замены значения в матрице.

2️⃣ Создайте класс Coordinates с двумя полями:

x — индекс строки,

y — индекс столбца.

3️⃣ Реализуйте метод:

static int[][] transformMatrix(int[][] matrix, MatrixTransformer transformer), который применяет логику трансформации к каждой ячейке матрицы и возвращает новую матрицу.

4️⃣ Создайте перечисление FlipDirection с двумя значениями: HORIZONTAL и VERTICAL, которое определяет направление переворота матрицы.

5️⃣ Реализуйте метод:

static int[][] flipMatrix(int[][] matrix, FlipDirection flipDirection), который переворачивает матрицу в зависимости от переданного значения FlipDirection.

Пример вывода:
Оригинальная матрица:
[1, 2]
[3, 4]
Горизонтальный переворот:
[2, 1]
[4, 3]
Вертикальный переворот:
[3, 4]
[1, 2]

Почему эта задача полезна?
Потому что в этом примере мы учимся декомпозировать большую и абстрактную задачу, используя при этом собственный функциональный интерфейс.
