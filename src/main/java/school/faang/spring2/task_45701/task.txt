Условие задачи:
Вы работаете разработчиком в дата центре Google и создаёте систему сбора аналитики использования ресурсов дата-центра. Вам нужно сделать её достаточно гибкой и масштабируемой для будущих новых сценариев, поэтому вы решаете использовать лямбды и функциональные интерфейсы.

Требования:
1️⃣ Создайте интерфейс Calculator, который будет принимать два значения одного типа (например, Integer) и возвращать результат какой-либо операции (например, сложение или умножение).

2️⃣ Реализуйте метод calculate, который принимает список чисел и объект Calculator, который выполняет операцию над числами из списка.

public static int calculate(List<Integer> nums, Calculator calculator) throws IllegalArgumentException
3️⃣ Создайте два метода, которые используют calculate:

product — возвращает произведение всех чисел в списке;

sum — возвращает сумму всех чисел в списке.

Почему эта задача полезна?
Изучение того, как использовать функциональные интерфейсы и Java — отличный способ развить свои навыки в программировании. Это задание научит вас, как использовать функциональные интерфейсы. Главное преимущество заключается в том, что это позволяет вам писать код более эффективным и кратким способом. Создав класс calculator с функциями многократного использования, вы можете создавать более сложные программы с меньшим количеством строк кода. Кроме того, вы получите лучшее представление о том, как использовать библиотеки Java.
