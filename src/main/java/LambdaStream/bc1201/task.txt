Условия задачи
Описание: Вы являетесь разработчиком приложения Telegram и работаете над функцией,
которая позволяет пользователям настраивать обработку входящих сообщений с помощью фильтров.
Вам необходимо создать систему, которая будет применять набор пользовательских фильтров к
входящим сообщениям с использованием собственного функционального интерфейса.

Цель: Создайте собственный функциональный интерфейс MessageFilter и класс MessageProcessor,
который будет использовать список объектов MessageFilter для обработки сообщений.

Требования:

Создайте функциональный интерфейс MessageFilter с методом filter, который принимает строку
(сообщение) и возвращает boolean (применяется фильтр или нет);

Создайте класс MessageProcessor с методом processMessage;

Метод processMessage должен принимать входящее сообщение и список объектов MessageFilter для обработки сообщения;

Создайте несколько фильтров с помощью лямбда-выражений и добавьте их в список;

Протестируйте вашу систему, обрабатывая несколько входящих сообщений с разными фильтрами.
