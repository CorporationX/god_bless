Описание

Условие задачи:
Вам предоставлены данные о пользователях и их действиях на платформе социальной сети. Задача — проанализировать эти данные и выявить наиболее активных пользователей и популярные темы обсуждений с помощью Stream API.

Данные представлены в виде списка объектов UserAction, у которых есть следующие поля:

id юзера;

name юзера;

actionType (возможные значения: "post", "comment", "like", "share");

actionDate — дата действия;

content — текст поста или комментария.

Этот список будет передаваться в ваши методы. Вам нужно лишь описать работу с ним. Самостоятельно его создавать не требуется.

Требуется реализовать:
1. Топ-10 самых активных пользователей (по количеству действий: посты, комментарии, лайки и репосты).

Метод должен возвращать список из 10 пользователей, которые совершили наибольшее количество действий.

2. Топ-5 наиболее популярных тем обсуждения (по количеству упоминаний хештегов в постах и комментариях).

Метод должен возвращать список из 5 самых популярных хэштегов (слова, начинающиеся с символа #).

3. Топ-3 пользователей, которые оставили наибольшее количество комментариев в последний месяц.

Метод должен возвращать 3 пользователей, которые оставили больше всего комментариев за последний месяц.

4. Вычислить процент для каждого типа действий (посты, комментарии, лайки и репосты).

Метод должен возвращать процентное соотношение каждого типа действия среди всех действий.
