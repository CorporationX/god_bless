Система трекинга
??? ??? (Дополнительная задача)


Условия задачи
Вам поручено создать систему управления транспортом, которая поможет отслеживать местоположение транспортных средств и определять их статус в реальном времени. Используя GPS-трекеры, транспортные средства будут передавать данные о своем местоположении в центральную систему, которая будет обновлять эту информацию и следить за их статусом (свободен или занят). Система будет работать в многопоточной среде.

Требования:
1?? Создайте класс Location, который представляет местоположение транспортного средства. Поля:
latitude — широта (тип double),
longitude — долгота (тип double).
Класс должен быть immutable для безопасности в многопоточной среде

2?? Создайте класс Vehicle, представляющий транспортное средство. Поля:
id — уникальный идентификатор транспортного средства,
status — статус транспортного средства (свободен или занят),
location — ссылка на объект Location.

Методы:
updateStatus(String newStatus) — обновляет статус транспортного средства,
updateLocation(Location newLocation) — обновляет местоположение транспортного средства.

3?? Создайте класс TransportManagementSystem, который хранит информацию о всех транспортных средствах.
Используйте ConcurrentHashMap для хранения объектов Vehicle. Что будет ключом в такой мапе?
Создайте метод addVehicle(Vehicle vehicle) — добавляет транспортное средство в систему
Метод updateVehicleLocation(String vehicleId, Location newLocation) — обновляет местоположение транспортного средства
И метод getVehicleInfo(String vehicleId) — возвращает информацию о транспортном средстве по его идентификатору.

4?? Создайте класс VehicleTracker, который отслеживает местоположение и статус транспортных средств в реальном времени. Используйте ScheduledExecutorService для периодического обновления местоположения транспортных средств.

5?? Протестируйте работу программы:
Создайте несколько транспортных средств и добавьте их в систему.
Отслеживайте их местоположение и статус в реальном времени, используя VehicleTracker.

Почему эта задача полезна?
Задача моделирует ситуацию, которая часто встречается в реальном мире — отслеживание местоположения транспортных средств в системе управления транспортом.

Это может быть полезно для компаний, занимающихся логистикой, транспортировкой грузов и пассажиров. Эта задача может быть отправной точкой для создания более сложных и специализированных систем, востребованных в реальном мире.

Например, можно добавить возможность отслеживания статуса транспортных средств (например, находится ли грузовик на загрузке, разгрузке или в пути), оповещение пользователей об изменении статуса, а также интеграцию с другими системами, такими как планирование маршрутов и определение оптимального времени доставки.

Ну и, конечно, прокачка навыков работы с многопоточностью и использование важных концепций Java, таких, как Immutable классы, volatile переменные, ConcurrentHashMap, и ScheduledExecutorService. Они важны для написания производительного и многопоточного кода.
