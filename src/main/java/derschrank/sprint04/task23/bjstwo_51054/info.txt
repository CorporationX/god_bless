Условия задачи
Пентагон разрабатывает систему обмена сообщениями между военными базами. Ваша задача — разработать приложение на Java, которое обеспечивает безопасный и быстрый обмен информацией между военными базами. Обмен сообщениями осуществляется с использованием симметричного шифрования AES, многопоточности и потокобезопасной очереди LinkedBlockingQueue.

Требования:
1?? Создайте класс EncryptionUtils, который будет содержать методы для шифрования и расшифровывания сообщений с использованием алгоритма AES.
AES-шифрование — алгоритм симметричного шифрования, для которого требуется один и тот же ключ для шифрования и расшифрования данных.
В таком случае, нам понадобится поле KEY — симметричный ключ шифрования (можно использовать константу).
Добавьте методы:
encrypt(String message) — принимает строку-сообщение и возвращает зашифрованную строку,
decrypt(String encryptedMessage) — принимает зашифрованную строку и возвращает расшифрованное сообщение.

2?? Создайте класс MilitaryBase, который представляет военную базу. Поля:
inbox — очередь сообщений для данной военной базы
running — флаг активности, контролирующий состояние объекта MilitaryBase (запущен/остановлен)
Класс должен реализовывать интерфейс Runnable. В методе run должна быть постоянная проверка очереди сообщений (если база активна, раз в секунду пробовать доставать сообщения из inbox и расшифровывать их с выводом в консоль).
Метод sendMessage(MilitaryBase destinationBase, String message) — зашифровывает сообщение и отправляет его в inbox другой базы,
а метод stop() — для остановки объекта MilitaryBase. Он останавливает проверку сообщений (меняет флаг running на false).

3?? Создайте класс App с методом main:
В методе создайте два экземпляра класса MilitaryBase,
Используйте Executors.newCachedThreadPool() для создания потоков, которые будут исполнять объекты MilitaryBase,
Отправляйте сообщения между базами и выводите результаты,
Остановите работу баз с помощью метода stop(),
Не забудьте вызвать shutdown для остановки всех потоков в ExecutorService.

Почему эта задача полезна?
Военные учреждения и службы безопасности часто используют системы передачи и обработки информации с использованием шифрования. Они также должны уметь работать в многопоточной среде для эффективной обработки большого количества данных.
Разработка безопасной системы передачи и обработки информации требует использования различных методов шифрования, в том числе и алгоритма AES, который был использован в данной задаче. Также, для обеспечения высокой производительности и эффективности работы системы, очередь LinkedBlockingQueue и многопоточность являются необходимыми компонентами.
