Условия задачи
В сеттинге «Игры престолов», семь королевств сражаются за железный трон. Ваша задача — написать программу на Java, используя CompletableFuture, которая асинхронно отправляет разведывательных воронов с сообщениями между королевствами. Если ворон не смог доставить сообщение, программа должна обрабатывать ошибку с помощью метода handle у CompletableFuture.

Требования:
1️⃣ Создайте класс Kingdom, который представляет королевство. Поля:
name — название королевства;
и метод sendMessage(Kingdom receiver) — метод, который отправляет сообщение из одного королевства в другое.

2️⃣ Реализуйте метод sendRaven(Kingdom sender, Kingdom receiver), который:
Асинхронно отправляет сообщение из королевства sender в королевство receiver.
Иногда вызывает исключение, имитируя недоставленное сообщение.
Возвращает сообщение об успешной доставке или бросает исключение.

3️⃣ Используйте метод handle у CompletableFuture, чтобы:
Обрабатывать успешные отправки сообщений.
Перехватывать исключения и обрабатывать недоставленные сообщения, выводя информацию об ошибке.

Почему эта задача полезна?
Обработка ошибок важна всегда. В любой системе, даже в Google Search всегда были и будут какие-то ошибки, исключения, нестандартные ситуации. Именно поэтому хороший разработчик всегда наперед думает об их обработке.

И то же самое справедливо для асинхронных задач. Они ведь тоже могут завершаться успешно, а могут с ошибками. Так вот у CompletableFuture есть удобные методы для обработки таких ситуаций!
