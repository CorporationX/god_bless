# Повернём матрицу (Дополнительно) lvl3

## Условия задачи

1. Представим, что перед нами стоит задача: уметь трансформировать квадратную матрицу (двумерный массив). Мы, как умные 
программисты, решили написать на это функциональный интерфейс `MatrixTransformer`, и написать метод `transformMatrix`, 
который его принимает, чтобы как-то преобразовать квадратную матрицу и вернуть новую.

```java
static int[][] transformMatrix(int[][] matrix, MatrixTransformer transformer)
```

2. `MatrixTransformer` имеет единственный метод, который принимает два числа (индексы ячейки), и возвращает так же 
2 числа. В Java нельзя вернуть сразу 2 результата из функции, поэтому нужно написать класс `Coordinates`, 
с 2-мя полями `x` и `y`.

3. Написать метод, который принимает `FlipDirection`, и в зависимости от него переворачивает квадратную матрицу.

```java
public enum FlipDirection {
HORIZONTAL,
VERTICAL,
}

static int[][] flipMatrix(int[][] matrix, FlipDirection flipDirection)
```

### Например матрица:


1 2  
3 4

При горизонтальном повороте примет вид:

2 1  
4 3

## Почему эта задача полезна?
Потому что в этом примере мы учимся декомпозировать большую и абстрактную задачу, используя при этом собственный 
функциональный интерфейс.
