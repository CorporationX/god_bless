# Потренируем StreamAPI 1

## Условия задачи
Напишите класс со статическими методами:
1. Найти сумму четных чисел в списке. На вход получаем список чисел, на выходе должны получать int;
2. Найти максимальный элемент в списке чисел;
3. Найти среднее значение чисел в списке;
4. Найти количество строк, начинающихся с определённого символа в списке строк;
5. Отфильтровать список строк и оставить только те, которые содержат определенную подстроку;
6. Отсортировать список строк по длине;
7. Проверить, все ли элементы списка удовлетворяют определённому условию;
8. Найти наименьший элемент в списке, который больше заданного числа;
9. Преобразовать список строк в список их длин.

## Почему эта задача полезна?
Решение задач на использование стримов предоставляет отличную возможность улучшить свои навыки программирования 
и логического мышления.

Стримы позволяют написать более компактный и выразительный код, который легче читать и поддерживать. 
Использование стримов позволяет избежать большинства проблем, связанных с изменяемыми состояниями, 
что делает код более безопасным и предсказуемым.
