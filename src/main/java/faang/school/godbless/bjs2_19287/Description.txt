Есть серверная - просто пустое помещение.
В серверную можно добавлять сервера.
У сервера два состояния:
  active = false - когда просто поставили, но не включили; не потребляет энергию
  active = true - когда включили; начал потреблять дефолтное значение энергии (буз нагрузки)

У сервера есть параметры:
  load - текщая нагрузка
  maxLoad - макимальная нагрузка (устанавливается вл время добавления в серверную)
  utilization - какой процент от максимальной нагрузки используется

Сервер потребляет энергию.
Формула расчета:
  DEFAULT_ENERGY_CONSUMPTION - дефолтное потребление, просто когда включён
  +
  load * coefficient
    где
      load - на каждую доп единицу нагрузки поребляется доп единица энергии
      coefficient - в зависимости от utilization:
        0-24 -> 1
        25-49 -> 1.2
        50-74 -> 1.5
        75-100 -> 2.5
      (то есть если сервер загружен на 100 процентов, то сильно греется и нужно включать вентиляторы ещё сильнее)

Логика добавления нагрузки.
Идём по всем серверам и для каждого проверяем:
  если currentLoad + requestedLoad < maxLoad - то добавляем
Если прошли через все сервера и ни для одного не добавили - то создаем новый.

Логика оптимизации.
Если у сервера:
  utilization = 0-24 - то он работает вхолостую (потребляет дефолтную энергию, а ничего не обрабатывает)
    то
      load этого сервера лучше перекинуть на другой, а этот сервер потушить
      либо
      дозаполнить чтобы utilization была 25 - 74 (ALLOWED_UTILIZATION_INTERVAL)
  utilization = 75-100 - то он работает на пределе, греется, изнашивается, ппоребляет энергию
    то
      надо load перебросить на другие сервера, и чтобы utilization на текущем сервере остался 49 (оптимальное значение)
        49 (оптимальное значение) - значение взято с головы; что-то среднее - и не простаивает и не перегружен
        причем переброс будет таким образом, чтобы на перебрасываемом сервере utilization была 25 - 74
Перебрасывание загрузки будет атомарно. То есть если нужно перебросить 40 - то все 40 перебрасываются на другой сервер,
а не размазываются по нескольким.
