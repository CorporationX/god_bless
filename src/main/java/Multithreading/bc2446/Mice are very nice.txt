Условия задачи
Том и Джерри живут в огромном доме. У них есть задача — собрать всю еду в доме. Нужно распараллелить
выполнение этой задачи на несколько потоков с использованием newScheduledThreadPool.

Класс House содержит список всех комнат и еды, которая есть в доме. Нам нужно создать пять потоков,
каждый из которых будет собирать еду из двух комнат, запускаясь с интервалом в 30 секунд.
огда все потоки закончат работу, программа должна вывести сообщение «Еда в доме собрана!».

Создайте класс House, который будет содержать список комнат и еды, а также метод для сбора еды из комнат.

В классе House создайте метод main, который будет запускать пул потоков и назначать каждому потоку
задачу для сбора еды из двух комнат.

Создайте класс Room, который будет содержать список еды в комнате.

Создайте класс Food, который будет описывать еду.

В методе main() создайте объект класса House и заполните его списки комнат и еды.

Создайте пул потоков с помощью метода Executors.newScheduledThreadPool(5).

В цикле for запустите каждый поток в пуле с интервалом в 30 секунд, используя метод executor.schedule() и
передавая ему лямбда-выражение, которое вызывает метод collectFood() из объекта house.

В методе collectFood() из объекта house, соберите еду из двух комнат и добавьте её в список собранной еды.

Удалите собранную еду из списков еды в комнатах.

После выполнения всех задач, закройте пул потоков и выведите сообщение о том, что еда собрана.
