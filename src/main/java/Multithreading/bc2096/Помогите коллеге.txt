Условия задачи
На обеденном перерыве женщина из кадров попросила вас написать маленькую программу,
которая из огромного списка людей достанет только места их работы…

Создайте класс Person с полями name, surname, age, workplace.

Создайте класс PersonNamePrinter, который реализует интерфейс Runnable.
Здесь нужно выводить в консоль имя каждого человека из списка.

В методе main класса Main cоздайте список объектов класса Person.
Этот список будет содержать информацию о 10 000 людей. Не нужно заполнять поля реальными данными,
как вариант, можно использовать для заполнения индекс элемента.

В методе main класса Main создайте объект класса ExecutorService с помощью метода newFixedThreadPool класса Executors.
Здесь нужно решить, на сколько потоков вы хотите разделить ваш список.

В методе main класса Main разбейте список на несколько частей.
Количество частей должно соответствовать количеству потоков в пуле.
Для каждой части списка создайте объект класса PersonNamePrinter.

Для каждого объекта PersonNamePrinter вызовите метод submit(), передавая ему объект в качестве аргумента.

Вызовите метод shutdown() объекта ExecutorService для завершения работы пула после того, как все задачи будут выполнены.

В методе main ожидайте завершения работы всех потоков, используя метод awaitTermination() объекта ExecutorService.

Выведите сообщение о завершении работы программы в консоль.
