Условия задачи
Король Роберт Баратеон собирается провести огромный турнир в Королевской Гавани.
В турнире будет участвовать множество рыцарей, и каждый рыцарь должен пройти через серию испытаний,
чтобы показать свою силу и мастерство. Король Роберт решил использовать ExecutorService для распараллеливания
выполнения этих испытаний — это вы ему подсказали, конечно. Ну и писать, значит, тоже вам…

Создайте класс Knight, который будет представлять рыцаря. Объявите поля name типа String и trials типа List<Trial>,
которое будет содержать список испытаний. Создайте конструктор, который принимает на вход имя рыцаря.

Реализуйте метод addTrial(Trial trial), который будет добавлять испытание в список испытаний.

Реализуйте метод startTrials(), который будет запускать испытания в отдельных потоках, используя пул потоков.

Создайте класс Trial, который будет представлять одно испытание. Объявите поля knightName типа String и
trialName типа String, которые будут содержать имя рыцаря и название испытания соответственно.

Реализуйте метод run(), который будет выполнять испытание.

В методе main() класса King создайте два объекта класса Knight и добавьте два испытания для каждого из них.

Создайте объект класса ExecutorService с помощью метода Executors.newFixedThreadPool(2)..

Добавьте задачи для исполнения в пул потоков с помощью метода executorService.submit()..

Закройте пул потоков, после выполнения всех задач, используя метод executorService.shutdown()..
