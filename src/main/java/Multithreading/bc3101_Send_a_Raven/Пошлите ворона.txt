Условия задачи
Задача: В сеттинге «Игры престолов», семь королевств сражаются за железный трон.
Ваша задача: написать программу на Java, используя CompletableFuture,
которая асинхронно отправляет разведывательных воронов с сообщениями между королевствами.
Если ворон не смог доставить сообщение, то программа должна обрабатывать эту ситуацию,
используя метод handle у CompletableFuture.

Пример классов:

Класс Kingdom — описывает королевство. name — название королевства;
sendMessage(Kingdom) — отправляет сообщение другому королевству.

Основные шаги:

Создайте несколько объектов класса Kingdom для разных королевств.

Напишите асинхронный метод sendRaven(Kingdom, Kingdom), который отправляет ворона с
сообщением из одного королевства в другое. Иногда этот метод будет вызывать исключение,
имитируя недоставленное сообщение. Метод может возвращать сообщение об успешной доставке
или кидать исключение. Но вообще вы можете написать здесь всё, как вам угодно.

Используйте метод handle у CompletableFuture для обработки успешной доставки сообщения
или возникших исключений.
