Вы разрабатываете Telegram-бот, который должен обрабатывать
сообщения от пользователей и отправлять ответы.
Однако, вы хотите ограничить количество запросов к API Telegram
в единицу времени, чтобы не получить блокировку от серверов
Telegram (например, разрешить не больше 5 запросов в секунду).

Создать класс TelegramBot с приватными полями REQUEST_LIMIT,
requestCounter и lastRequestTime.

Создать конструктор класса, который будет инициализировать
поля requestCounter и lastRequestTime.

Создать метод sendMessage, который будет принимать сообщение
и отправлять его через API Telegram (ну, как будто бы).

В начале метода отправки сообщения получить текущее время и
вычислить время, прошедшее с момента последнего запроса.

Если прошедшее время меньше секунды, увеличивать requestCounter на 1,
и если он становится больше REQUEST_LIMIT, то подождать до конца секунды.

Если прошедшее время больше или равно секунде, то обнулить
requestCounter и lastRequestTime.

Отправить сообщение через API Telegram (ну, как будто бы).

В методе main класса Main запустить несколько потоков для проверки.
Cообщить, что задача выполнена.
