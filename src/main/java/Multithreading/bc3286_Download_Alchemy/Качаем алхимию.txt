Условия задачи
Задача: В сеттинге "The Elder Scrolls", вы работаете с алхимиком,
который собирается создать несколько зелий. Ваша задача: написать
программу на Java, используя CompletableFuture и Stream API,
которая асинхронно собирает ингредиенты для каждого зелья и
возвращает общее количество собранных ингредиентов.

Пример классов:

Класс Potion — описывает зелье. name — название зелья.
requiredIngredients — количество требуемых ингредиентов
для создания зелья.int

Основные шаги:

Создайте список объектов класса Potion для разных зелий.

Напишите асинхронный метод gatherIngredients(Potion),
который собирает ингредиенты для зелья. По сути, просто ждёт
какое-то время, а затем возвращает количество требуемых
ингредиентов для переданного зелья.

Используйте Stream API для создания списка CompletableFuture,
который асинхронно собирает ингредиенты для каждого зелья в списке.

Используйте метод allOf у CompletableFuture для ожидания
завершения сбора всех ингредиентов.

Выведите общее количество собранных ингредиентов.
Будьте осторожны здесь! Нужно, чтобы сложение было thread-safe!
