Условия задачи
У Walmart есть несколько касс, каждая из которых обслуживает покупателей.
Каждый покупатель — это отдельный поток. Напишите программу, которая использует наследование от класса Thread,
чтобы создать классы, представляющие кассы в Walmart.

Когда покупатель приходит в Walmart, он выбирает случайную кассу и становится в очередь.
Покупатель ждёт, пока кассир закончит обслуживание текущего покупателя, и только после этого начинает обслуживаться сам.

Создайте класс CashierThread, который наследуется от класса Thread.

В классе CashierThread объявите поля cashierId (идентификатор кассира) и customerItems (массив товаров покупателя). Создайте конструктор для класса CashierThread.

В методе run() класса CashierThread реализуйте логику обработки товаров покупателя кассиром.

В методе main() создайте двумерный массив customers, где каждый элемент массива представляет список товаров для каждого покупателя.

Создайте массив cashiers для хранения потоков кассиров. Каждый элемент массива cashiers должен соответствовать одному покупателю из массива customers.

В цикле for создайте объекты класса CashierThread и добавьте их в массив cashiers.

В цикле for запустите все потоки кассиров из массива cashiers с помощью метода start().

После выполнения всех потоков кассиров в методе run() каждого потока, выведете на экран информацию о количестве обработанных товаров и общей стоимости.
